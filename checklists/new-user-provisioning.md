# Checklist: new user provisioning

- [ ] Request approved (HR/manager) with role + required access.
- [ ] Account created in the **correct OU**.
- [ ] UPN / display name / temp password set; **"change at next logon"** checked.
- [ ] Added to the **role's security groups** (mirror a peer).
- [ ] Mailbox / license handled (hybrid → M365 side).
- [ ] First login validated; required resources reachable.
- [ ] Granted access **documented** for future offboarding.
