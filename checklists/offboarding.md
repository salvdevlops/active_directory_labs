# Checklist: user offboarding

- [ ] **Disable** the account (don't delete immediately — preserve for audit/handover).
- [ ] Reset password / revoke sessions so it can't be used.
- [ ] Remove from sensitive groups (or document why retained briefly).
- [ ] Handle mailbox (convert to shared / delegate) per policy.
- [ ] Reassign or archive owned data/resources.
- [ ] Reclaim licenses and hardware.
- [ ] Schedule deletion after the retention window.
