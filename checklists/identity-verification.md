# Checklist: caller identity verification (do this FIRST)

- [ ] Full name **and** a second identifier (employee ID / username).
- [ ] An org-approved verification step (callback, manager confirm, MFA, security question).
- [ ] Confirm the request matches the person's role (is this access reasonable for them?).
- [ ] **Record** the verification you performed in the ticket.
- [ ] Never reset or grant access on an unverified request — this is the #1 social-engineering entry point.
